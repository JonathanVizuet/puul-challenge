# Puul Challenge 

Repositorio central del challenge técnico para Puul. Este es un proyecto monorepo que contiene tanto la infraestructura del servidor (API) como la interfaz de usuario (Frontend).

---

## Documentación Detallada
Toda la lógica de negocio, diagramas de base de datos, requerimientos y documentación técnica se encuentran centralizados en nuestra documentación de Notion:

https://www.notion.so/Challenge-Back-End-Puul-30e83ffea94880d1abb6c976d1bc82e0?source=copy_link

---

## Estructura del Proyecto

Este repositorio se divide en dos módulos principales:

* `puul-api/`: Backend construido con NestJS y PostgreSQL. Utiliza TypeORM para la gestión de datos.
* `puul-frontend/`: Cliente construido con Angular y TypeSrcipte.

---

## Correr proyecto

Para ejecutar el proyecto localmente, sigue las instrucciones detalladas en cada carpeta:

1.  Configuración del Backend: Dentro de la carpeta 'puul-api' se encuentra un README para la configuración del archivo `.env` y levantamiento de la base de datos.

2.  Configuración del Frontend: Dentro de la carpeta 'puul-frontend' se encuentra un README para instalar las dependencias de la interfaz.

---

## Requisitos Globales

Se debe de tener instalado:
* Node.js
* PostgreSQL (corriendo localmente para el backend)
